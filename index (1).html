📋 Setup Instructions

Step 1: Add Apps Script (3 min)

  1. Open your Google Sheet
  2. Extensions → Apps Script
  3. Delete default code
  4. Paste the script (see below)
  5. Save (Ctrl+S)

Apps Script Code:

function doGet() { const ss = SpreadsheetApp.getActiveSpreadsheet(); const data = { membership: getSheetData('membership_revenue'), sponsorship: getSheetData('sponsorship_pipeline'), anniversary: getSheetData('anniversary_events'), board: getSheetData('board_roles'), revenue: getSheetData('revenue_summary') }; return ContentService.createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); } function getSheetData(sheetName) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(sheetName); if (!sheet) return []; const data = sheet.getDataRange().getValues(); const headers = data[0]; return data.slice(1).map(row => { const obj = {}; headers.forEach((h, i) => obj[h] = row[i] || ''); return obj; }); }

Step 2: Deploy (2 min)

  1. Click "Deploy" → "New deployment"
  2. Click gear icon → Select "Web app"
  3. Description: "NFBPA Dashboard API"
  4. Execute as: "Me"
  5. Who has access: "Anyone"
  6. Click "Deploy"
  7. Copy the Web app URL

Step 3: Configure Dashboard (1 min)

  1. Open this HTML file in text editor
  2. Find line: const APPS_SCRIPT_URL = 'YOUR_URL_HERE';
  3. Paste your Web app URL
  4. Save file

✅ Done!

Open dashboard in browser → Data loads automatically
